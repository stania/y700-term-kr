# i3 단축키 목록

> `$mod` = **Alt (Mod1)**

## 앱 실행

| 단축키 | 동작 |
|--------|------|
| Alt+Ctrl+W | wezterm 새 창 |
| Alt+Ctrl+I | kitty 새 창 |
| Alt+Ctrl+D | rofi 앱 런처 |
| Alt+Ctrl+R | rofi 앱 런처 (대체) |
| Win+R | rofi 앱 런처 (대체) |

## 창 관리

| 단축키 | 동작 |
|--------|------|
| Alt+Ctrl+Shift+Q | 현재 창 닫기 |
| Alt+Ctrl+F | 전체화면 토글 |
| Alt+Ctrl+S | 레이아웃: 스태킹 |
| Alt+Ctrl+E | 레이아웃: 분할 토글 |

## 포커스 이동

| 단축키 | 동작 |
|--------|------|
| Alt+Ctrl+H | 왼쪽 창으로 |
| Alt+Ctrl+J | 아래 창으로 |
| Alt+Ctrl+K | 위 창으로 |
| Alt+Ctrl+L | 오른쪽 창으로 |

## 창 이동

| 단축키 | 동작 |
|--------|------|
| Alt+Ctrl+Shift+H | 창을 왼쪽으로 |
| Alt+Ctrl+Shift+J | 창을 아래로 |
| Alt+Ctrl+Shift+K | 창을 위로 |
| Alt+Ctrl+Shift+L | 창을 오른쪽으로 |

## 워크스페이스

| 단축키 | 동작 |
|--------|------|
| Alt+Ctrl+1 | 워크스페이스 1 이동 |
| Alt+Ctrl+2 | 워크스페이스 2 이동 |
| Alt+Ctrl+3 | 워크스페이스 3 이동 |
| Alt+Ctrl+Shift+1 | 현재 창 → 워크스페이스 1 |
| Alt+Ctrl+Shift+2 | 현재 창 → 워크스페이스 2 |
| Alt+Ctrl+Shift+3 | 현재 창 → 워크스페이스 3 |

## 시스템

| 단축키 | 동작 |
|--------|------|
| Alt+Ctrl+Shift+E | i3 종료 |
| Alt (드래그) | 플로팅 창 이동 |

## 자동 시작

- **CapsLock → Ctrl** 리매핑 (`setxkbmap ctrl:nocaps`)
- **fcitx5** 한국어 입력기
- **wezterm** → 워크스페이스 1 자동 배정
